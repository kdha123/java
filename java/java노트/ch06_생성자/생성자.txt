생성자
1. 클래스이름과 동일한이름으로 기재한다.
2. 리턴형이 없다.
3. 목적: 필드초기화(객체초기화)
4. 오버로드가능 , 인수를 갖는다.
5. 객체생성시 바로 호출된다.
    new Test();//객체생성,생성자 호출.

6. 생성자를 넣지 않으면
    디폴트 생성자가 자동으로 만들어진다. 

7. 인자 있는 생성자를 넣으면
    디폴트 생성자가 만들어지지 않는다.

★this : 자신의 객체 
   this(): 자신의 클래스 내의 생성자 호출 ,생성자 내의 첫줄에 기재 

   super : 상위 클래스 객체 
★super() : 상위 클래스 내의 생성자 호출,생성자 내의 첫줄에 기재

 this()와 super()는 동시에 사용 못함
